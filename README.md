# Mini-Rogue

Description: 



Controls/Game Info:

'h' = Left
'j' = Up
'k' = Down
'l' = Right

"i' = Check Inventory 



'g' = Pick Item Up(hover on top of item and press 'g')


Characters:

'@' = You
'G'= Goblin
'S' = Snake Woman 

Objects:

'>' = Staircase
')' = Weapon
'?' = Scroll
